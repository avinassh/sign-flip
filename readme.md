# SignFlip

Using a [minimal approach](https://www.reddit.com/r/shittyprogramming/comments/7n35m7/how_to_flip_the_sign_of_a_number_a_minimal/) to flip the sign of a number.

## Install 

```
npm install sign-flip
```

## Usage

```
signFlip(666)
signFlip(-666)
```
